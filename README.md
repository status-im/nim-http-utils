# Installation

```
nimble install https://github.com/status-im/nim-http-utils
```
or
```
git clone https://github.com/status-im/nim-http-utils
```

## Requirements

* Nim 0.17.3 and up

## Documentation

To get documentation please run
```
nim doc httputils.nim
```
